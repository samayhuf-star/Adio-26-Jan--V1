# 14-Jan-Adiology

Adiology - Medical Imaging Analysis System

## Overview

Adiology is a simple medical imaging analysis system designed to manage and process medical images such as X-rays, CT scans, and other radiological data.

## Features

- Add and manage medical images
- Associate images with patient IDs
- Perform basic image analysis
- List and retrieve images from the system

## Installation

```bash
# Clone the repository
git clone https://github.com/samayhuf-star/14-Jan-Adiology.git
cd 14-Jan-Adiology

# Install dependencies (if any)
pip install -r requirements.txt
```

## Usage

### Basic Usage

```bash
# Run the application
python3 adiology.py
```

### Demo Mode

```bash
# Run with demo data
python3 adiology.py demo
```
